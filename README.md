# 📝 Sistema de Formulários Web

Este repositório contém um conjunto de formulários interativos desenvolvidos em **HTML5** e estilizados com **CSS3**, focados em capturar diferentes tipos de dados de forma organizada e responsiva.

## 🚀 Páginas do Projeto

*   **Ficha Médica (`ficha_medica.html`)**: Layout estruturado para a recolha de dados de triagem, histórico de saúde e informações médicas sensíveis.
*   **Formulário de Cadastro Geral (`formulario.html`)**: Um formulário versátil configurado com campos essenciais de identificação (nome, e-mail, endereço, cidade, entre outros), ideal para registo de utilizadores ou clientes.
*   **Inscrição para Vestibular (`inscricao.html`)**: Interface focada no meio académico, projetada para a recolha de dados de candidatos, opções de cursos e informações de inscrição de estudantes.

## 🛠️ Tecnologias e Conceitos Aplicados

*   **HTML5 Semântico**: Utilização de tags estruturais adequadas e validação nativa de campos (como `type="email"`, `required`, etc.).
*   **Estilização Avançada com CSS3**:
    *   Arquitetura visual limpa com a paleta de cores `#2c3e50`.
    *   Técnicas de alinhamento vertical e posicionamento de elementos no topo dos contêineres.
    *   Tipografia tratada (`font-size: 32px` nos títulos principais) para garantir uma boa hierarquia de leitura.
